# FART QUEST — Voice-Over Recording Script (Phase 1)
**Voice:** Professor Whiffbeard — ancient wizard, magnificent beard, smells appalling. Pompous, warm, DELIGHTED by farts, always on Jarlath's side. Big theatrical energy.

## How to record
1. Use Voice Memos on your iPhone in a quiet room, phone about 20 cm from your mouth.
2. Record one memo per line below. **Rename each memo to its exact filename WITHOUT the .m4a part** (e.g. rename to `vo-welcome-01`) — Voice Memos adds .m4a automatically when exported.
3. Keep each clip under ~8 seconds (except the welcome, which can be ~15). One good take is plenty — imperfect is funnier.
4. When done, AirDrop/save ALL files into this folder on the Mac:
   `/Users/damiengartland/Desktop/Claude Work/Jarlath SEAG App/VO Recordings/`
5. Then run the placement prompt in `AUDIO_PLACEMENT.md` — a cheap-model session will install them into the app. **The app works fine with only SOME files recorded** — record Priority A first, add the rest whenever.

Feel free to improvise around any line — funnier beats accurate. The only rule: never genuinely mean; always end failure lines with fight-back energy.
